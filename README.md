# 📈 Smart Stock Analyzer

A real-time stock market analyzer built with Python, Streamlit & Plotly.

## 🚀 Features
- Real-time stock data from Yahoo Finance
- Interactive Price Trend Chart
- Candlestick Chart (OHLC)
- Buy / Sell Signals using Moving Averages (MA20 & MA50)
- Compare multiple stocks
- Supports both Indian (NSE) & US stocks
 - Analyzed real stocks like Tesla, Reliance & Apple
- Discovered Tesla dropped 24% in last 3 months!

## 🛠️ Tech Stack
- Python
- Streamlit
- Plotly
- yfinance
- Pandas

## 📊 How to Run
```bash
pip install streamlit yfinance plotly pandas
streamlit run dashboard.py
```

## 📌 Stock Symbol Examples
- Indian Stocks: RELIANCE.NS, TCS.NS, ZOMATO.NS
- US Stocks: TSLA, AAPL, GOOGL
